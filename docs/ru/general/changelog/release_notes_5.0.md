# Сведения о выпуске 5.0

## Введение

В **{{ productName }}** версии 5.0 от 10.12.2024 были реализованы реализованы перечисленные ниже новые возможности, улучшения и исправления по сравнению с версией 4.7.

Этот релиз включает важные улучшения и исправления, направленные на повышение стабильности, функциональности и удобства использования системы. Мы сосредоточились на развитии платформы, улучшении интерфейса, а также исправлении обнаруженных ошибок.

## Ключевые изменения

- **Конфигурация платформы**: расширены возможности настройки платформы, что делает систему более гибкой и адаптируемой.  
- **Диаграммы и элементы интерфейса**: оптимизирован процесс работы с диаграммами и элементами интерфейса, улучшена их визуализация и функциональность.  
- **Совместимость**: улучшена совместимость с различными системами и окружениями.  
- **Экспорт и резервное копирование**: повышена стабильность функций экспорта и резервного копирования.  
- **Расширенная фильтрация**: добавлена поддержка расширенной фильтрации (фасетного поиска) по атрибутам записей.
- **Экспорт и импорт версий через Excel**: введена поддержка экспорта и импорта версий приложения с помощью Excel.

## Новые возможности и улучшения

### Расширенная фильтрация (фасетный поиск)

- **Форма поиска**: добавлен новый тип форм, позволяющий гибко настраивать поиск по атрибутам записей.
- **Фильтрация данных**: реализована настройка логических операторов для фильтрации данных.
- **Фильтрация данных**: добавлена возможность сочетания фильтрации с помощью форм поиска и на уровне столбцов таблиц.  

- **Контекстное меню чекбоксов в таблицах**:  
   - Добавлена возможность выбора всех записей или только отображаемых записей в таблице с учетом настроенных фильтров, улучшая контроль над выборкой данных.

#### Сценарии использования

- **Расширенный поиск в таблицах и коллекциях**: пользователи могут настраивать фильтры по атрибутам и колонкам для точной выборки данных, что позволяет эффективно работать с большими объемами информации.  
- **Поиск по атрибуту типа "Запись"**: при множественном выборе записей добавлена возможность фильтрации и отображения только выбранных записей через специальную вкладку, упрощая процесс работы с записями.

#### Ограничения

- **Переключение типа формы**: невозможно переключить тип формы между "Публичной", "Внутренней" и "Формой поиска" без создания новой формы.  
- **Ограничения формы поиска**: в форме поиска не поддерживаются статичный текст, кнопки, вкладки и группы вкладок.

### Улучшенный глобальный поиск

- **Поиск по свойствам документа**: реализована возможность выполнения поиска по текстовым полям внутри документов.
- **Поиск по именам вложений**: поддержан поиск по названию вложенных файлов.
- **Поиск по содержимому вложений**: добавлена возможность поиска по текстовому содержимому файлов форматов `txt`, `csv`, `xml`, `docx`, `xlsx` и других.


Поддержка расширенного глобального поиска теперь охватывает документы и вложения, значительно расширяя возможности работы с данными.  

- **Поиск по свойствам документа**: позволяет находить данные внутри текстовых полей документа.  
- **Поиск по именам вложений**: упрощает доступ к вложенным файлам, поддерживая поиск по их названиям.  
- **Поиск по содержимому вложений**: работает с форматами "txt", "csv", "xml", "eml", "mhtml", "htm", "html", "docx", "xlsx", позволяя находить информацию даже в специфических типах документов.  
- **Работа с multivalue-документами и ревизиями**: корректно обрабатывает запросы, отображая оригинальное имя файла при нахождении совпадений в ревизиях.  
- **Доступность атрибутов типа "Документ" для поиска**: такие атрибуты теперь можно помечать как доступные для поиска, что расширяет их функциональность.  
- **Автоматическое обновление данных поиска**: после загрузки новых документов, ревизий или изменения атрибутов поиск корректно работает по как старым, так и новым данным.

### Диаграммы

Поддержка работы с диаграммами стала более удобной и функциональной благодаря новым возможностям и улучшениям.  

- **Выравнивание элементов**: реализована возможность выравнивания по горизонтали и вертикали. Теперь элементы можно выделять как с помощью клавиатуры, так и с использованием инструмента «лассо».  
- **Функция "привязки"**: упрощает размещение элементов на диаграммах, делая процесс редактирования более интуитивным.  
- **Улучшенный экспорт**: добавлена кнопка экспорта с поддержкой сохранения диаграмм в форматах PNG, SVG и BPMN.  
- **Типы диаграмм**: отображение круговых, линейных, столбчатых и других типов диаграмм доступно с обновлённым интерфейсом.  
- **Работа с элементами диаграмм**: усовершенствован процесс взаимодействия, включая ручное регулирование потоков и автоматическое выравнивание для оптимальной компоновки.

### Улучшения в работе с диаграммами

Для улучшения визуальной четкости и удобства создания диаграмм были внедрены новые функции, такие как автоматическое выравнивание элементов и ручное регулирование потоков. Эти изменения значительно повышают точность размещения элементов и настройку направлений потоков, создавая аккуратный и организованный вид диаграмм.

#### Основные возможности

- **Автоматическое выравнивание элементов**:  
   - При перемещении элементов система автоматически выравнивает линии границ и центров элемента с другими объектами. Линии выравнивания отображаются при совпадении и исчезают при дальнейшем перемещении. Это упрощает процесс упорядочивания элементов на диаграмме.  

- **Ручное регулирование стрелок потоков**:  
   - Стрелки потоков теперь можно регулировать вручную с помощью контрольных точек на каждом изгибе, что позволяет гибко изменять направление потока.  
   - Для прямых стрелок добавлены центральные точки для выравнивания и корректировки каждой половины потока, а новые точки появляются при добавлении изгибов.  

- **Указание потоков на элементы**:  
   - Потоки теперь можно направить не только на коннекторы, но и на элементы. Система автоматически определяет ближайший коннектор для корректного отображения потока.

- **Коннекторы по сетке**:  
   - Коннекторы выравниваются по сетке диаграммы, что устраняет искривление стрелок потоков и гарантирует их симметричное отображение.  

- **Встраивание элементов в потоки**:  
   - Возможность встроить элемент непосредственно в поток, что приводит к автоматическому разделению потока на две части: входящую и исходящую.

#### Преимущества

Эти улучшения значительно повышают удобство работы с диаграммами, позволяют быстрее и точнее строить диаграммы, а также обеспечивают более эстетичный и аккуратный вид конечного результата.

- **Линии привязки при масштабировании**: решены проблемы с отображением линий привязки при изменении масштаба диаграмм.

### Мобильное приложение

- **Дизайн страниц поиска**: скорректирован дизайн страниц поиска для соответствия утвержденным макетам, что улучшает визуальное восприятие в мобильном приложении.
