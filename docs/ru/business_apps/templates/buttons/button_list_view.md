# Просмотр списка кнопок в шаблоне

Список кнопок предусмотрен у [шаблонов][templates] всех типов.

1. Откройте шаблон.
2. Выберите вкладку «**Кнопки**».
3. Отобразится список кнопок в составе шаблона.
